psmm — next level
Proposed evolution of the design system. Everything below stays inside the current stack: Spectre markup, explicit 256-color names, keyboard only. Key capsules = [grey19 on salmon1]-style background markup; row highlight = per-row background style; the goto overlay is just a re-render with a small panel — no new dependency.
Keep: the discipline
Single palette source, key registry with tests, install ≠ update, alt-screen restore, too-small fallback, row x/n everywhere. This rigor is the system's best asset — v2 changes surfaces, not rules.
1 · Hints have no hierarchy
Today every key is the same salmon-on-grey pair; 20+ bindings always visible. v2: keys become capsules (reverse-video blocks your eye can target), split into two tiers — contextual verbs vs a dim persistent strip (2a).
2 · g becomes the goto layer
Eight screen-switch letters burn the alphabet and crowd the grid's third hint row. v2 moves ALL navigation under the existing g chord with a vimium/which-key overlay (2b) — same key everywhere, single letters stay reserved for verbs.
3 · Mode/Inst are config jargon
"InstallOnly / IfMissing" answers neither "what happens at startup?" nor "what may the gallery do?". v2 renames the columns startup and gallery, lowercases the values, and adds a one-line context sentence for the cursor row (2a).
4 · Help is a wall of text
One 96-line pager. v2 reuses the command-detail tab pattern that already exists (30-Commands.ps1): topic tabs, and the key reference becomes a grouped two-column layout (2c).
5 · Weak cursor, loud borders
A lone ">" plus a recolored name is easy to lose; bright #ccc borders shout louder than content. v2: full-row background highlight with a ▌ bar, borders dimmed to grey27, headers lowercase + dim (2a).
6 · Startup report ≠ the TUI
"> Name <" wrappers and raw ConsoleColor look like a different product. v2 gives the report the same tokens: state glyphs, aligned timings with bars, the slowest module called out (2d).
Identity
The  psmm  brand block in a one-line header bar gives every screen the same anchor + breadcrumb — the "where am I" affordance the current title line only hints at. Palette variants in 2g.
